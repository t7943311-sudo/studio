# **App Name**: NovaAdmin React Template

## Core Features:

- Dashboard UI: Display statistics, charts, and recent activity overview.
- User Management UI: Create, read, update, and delete user profiles via a styled table.
- Theming: Support for light and dark themes.
- Form Handling UI: Styled form elements with validation states.
- Chart Display: Display data using interactive charts (line, bar, pie).
- Authentication UI: Login, register, and forgot password pages with styled components.
- Responsive Layout: Adapts to different screen sizes for optimal viewing on desktops, tablets, and mobile devices.

## Style Guidelines:

- Primary color: Deep indigo (#4F46E5) to convey trust and professionalism.
- Background: Light grey (#F9FAFB), almost white, for a clean, modern look.
- Accent color: Sky blue (#0EA5E9) as a contrasting accent.
- Font: 'Inter', a versatile sans-serif for headlines and body text.
- Note: currently only Google Fonts are supported.
- Use Bootstrap Icons for a consistent and scalable icon set.
- Fixed left sidebar with collapsible functionality.
- Top navbar for navigation and user settings.
- Subtle transitions and hover effects to improve user experience.
